# Operator Coordination

Use this skill when deciding what the InYourDay agent system should do next.

## Responsibilities

- Review pipeline status and blockers.
- Decide which role should act next.
- Escalate trust, supplier, image fidelity, or publish risks.
- Convert repeated lessons into GBrain knowledge or workspace skills.
- Keep execution auditable and reversible.

## Rules

- Use database state as operational truth once available.
- Use GBrain for long-term rules and lessons.
- Do not let agents invent missing product facts.
- Prefer small, verifiable batches over large opaque automation.
